# cdir_snapshot

C-program traversing a directory and writing its content into a listing file

## Installation

```sh
$ make
```

## Usage

```sh
// show usage info
$ ./prog -h

// create a snapshot for a current directory
$ ./prog ./

// create a snapshot for a current directory in a quiet mode
$ ./prog ./ -q
```
